# AUTHODOCS – Secure Student Document Management System

## 📌 Overview

AUTHODOCS is a full-stack web application designed to streamline the process of requesting, managing, and retrieving student documents in a centralized system. It allows students to request documents and enables staff to review, approve, or reject requests efficiently.

---

## 🚀 Features

* User authentication (Student & Staff roles)
* Secure document request system
* Admin approval/rejection workflow
* Real-time request status tracking
* Document download functionality
* Centralized data management

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

---

## ⚙️ How It Works

1. Students log in and submit document requests.
2. Staff/Admin reviews the requests.
3. Requests are approved or rejected.
4. Approved documents are made available for download.
5. Students can track request status in real-time.

---

## 💻 Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/authodocs.git
```

2. Navigate to project folder:

```
cd authodocs
```

3. Install dependencies:

```
npm install
```

4. Start the server:

```
node server.js
```

5. Open in browser:

```
http://localhost:3000
```
